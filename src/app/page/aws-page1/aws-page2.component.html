Ephemeral Ports

  • Short-lived transport protocol used for IP communication
  • Above the "Well-know" IP ports (above 1024)
  • Sysnamic Ports
  • Suggested range is 49152 to 65535 but...
    • Linux kernals generally use 32568 to 61000
    • window platform default from 1025
  • NACL and Security Group implication.
